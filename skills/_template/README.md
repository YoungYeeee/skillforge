---

## How to Use These Prompts

1. Copy the prompt from the `.md` file
2. Paste into your preferred AI (Claude, GPT-4, Gemini, etc.)
3. Follow the AI's instructions — it will guide the session
4. After each session, reflect: *What did I learn? What's still unclear?*
5. Repeat exercises until you no longer need to look at the prompt

**Important:** If the AI does the thinking for you, you're using it wrong. Push back, ask it to question you instead of answer you.
