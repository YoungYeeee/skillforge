# Prompt：自我评估 —— 批判性思维

**目的：** 在开始学习前建立基准。帮助 AI 根据你的实际水平调整辅导方式。

**使用时机：** 第一天，在使用本模块其他 Prompt 之前。

**预计时长：** 20–30 分钟

---

## Prompt 内容

```
我想评估自己目前的批判性思维水平，以便更有效地学习这项技能。

请和我一起进行一次结构化的自我评估。现在不要讲课或教我内容——只做评估。

请按以下方式进行：

1. 每次只问我一个问题，等我回答后再问下一个。
2. 从宽泛的问题开始，根据我的回答逐步深入。
3. 重点关注以下四个方面：
   - 发现假设：我能发现论点中默认的前提吗？
   - 钢人论证：我能公平地呈现对立观点吗？
   - 因果推理：我会把相关性和因果性混淆吗？
   - 自我觉察：我能意识到自己的偏见吗？
4. 问完 6–8 个问题后，给我一个诚实的评估：
   - 我哪些方面理解清晰
   - 我的理解在哪里存在盲点或误区
   - 我在每个方面处于什么水平（新手 / 发展中 / 熟练）
   - 我应该优先专注于哪个方面

请从这个问题开始："请讲一件你改变了重要想法的事情。是什么让你改变了想法？"

如果我想不出例子，请问："用你自己的话说，'批判性思维'对你意味着什么？当一个人具备这种能力时，具体表现是什么样的？"

记住：你现在的任务是了解我在哪里，而不是教我。
```

## 学习者须知

- **诚实作答。** 高估自己会导致 AI 设定错误的学习起点。
- 如果 AI 在评估结束前开始教学，提醒它："请先完成评估。"
- 保存这份评估结果，毕业时你将把它与结业评估对比，看看自己的进步。
