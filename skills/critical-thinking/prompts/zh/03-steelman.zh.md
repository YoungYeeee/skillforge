# Prompt：钢人论证练习

**目的：** 培养以最强形式呈现对立观点的能力——这是真正理智诚实和有说服力的推理的前提。

**使用时机：** 完成假设识别练习后使用。每当遇到你强烈反对的观点时，回来做这个练习。

**预计时长：** 每次 30–60 分钟

---

## 背景

**稻草人论证**是对对立观点的歪曲，让它更容易被推翻。
**钢人论证**是以最强形式呈现对立观点——通常比真正持有这一观点的人表达得还要有力。

钢人论证很难，因为它要求你暂时进入一个你拒绝接受的世界观。但它也是你能拥有的最有价值的思维技能之一：你无法有效地反驳一个你不真正理解的立场。

---

## Prompt 内容

```
我正在练习钢人论证——以最强形式呈现对立观点。

我的角色：我会分享一个我不认同的立场，并尝试对它进行钢人论证。
你的角色：推动我让钢人论证更有力。不要接受一个弱版本。

辅导方式：
1. 在我给出钢人论证尝试后，用 1–5 分来评分（1 = 稻草人，5 = 比大多数支持者说得还好）
2. 问我："真正持有这一观点的人会觉得被准确呈现了吗？"如果不会，我的版本哪里不足？
3. 通过向我提问（而不是直接告诉我），帮我找到支持这一立场的最有力证据、最有力的道德直觉和最有力的推理。
4. 持续推动，直到我的钢人论证达到至少 4/5。

我不认同的立场：
[在这里写出立场——要具体，不要模糊]

我最初的钢人论证尝试：
[在这里写你的尝试——不要去查资料，从你自己的理解开始]
```

---

## 练习选题建议

选一个你有真实观点的话题：

- 你反对的一项政策
- 你不喜欢的一种管理风格
- 你认为有问题的饮食/生活方式选择
- 你认为有害的一项技术
- 你认为是错误的一个历史决定

**提示：** 你的情绪反应越强烈，这个练习越有价值。对你轻微反对的事情进行钢人论证很容易。真正的训练发生在你容忍极限的边缘。

---

## 课后记录

1. **我的钢人论证得分（1–5）：**

2. **对立观点中我之前没有考虑过的最有力论点：**

3. **这个练习改变了我对这个话题的看法吗？为什么？**

4. **我对这个立场的原有理解中有什么是错的：**

---

## 公平性检验

写完钢人论证后，找一个真正持有对立观点的人（或使用以下 Prompt）：

```
这是我尝试呈现一个我不认同的观点。请告诉我：
1. 这准确地呈现了你的立场吗？
2. 有什么遗漏或误解？
3. 你的立场中最有力的论点，我没有包含的是什么？

我的钢人论证：[粘贴你的钢人论证]
```

如果对方确认这是准确的——或者如果扮演支持者的 AI 会接受它——你就通过了。
