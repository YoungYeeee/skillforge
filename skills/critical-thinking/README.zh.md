# 批判性思维

> 评估主张，发现隐含假设，通过推理得出更好的结论。

**难度：** 入门 → 中级
**预计时长：** 2–3 周，共 3–5 小时
**语言：** English · 中文

---

## 学习目标

完成本模块后，你将能够：

- [ ] 识别任何论点中的显性和隐性假设
- [ ] 对对立观点进行钢人论证（比支持者自己表达得更公平）
- [ ] 区分日常主张中的相关性与因果性
- [ ] 在不借助 AI 的情况下，提出能暴露弱推理的问题
- [ ] 识别自己思维中常见的认知偏见

---

## 前置要求

无。这是一个基础模块。

---

## 为什么这项技能重要

大多数人从不审视自己为什么相信某件事。我们从媒体、社交圈和情绪中吸收信念，然后事后才找理由来支撑它。批判性思维的实践就是逆转这一过程——从证据出发，通过推理得出结论。

这项技能是所有其他技能的基础：更好的写作、更好的决策、更好的技术工作。没有它，你可以表达流畅，但系统性地犯错。

AI 让这一点变得既更容易，也更危险。更容易，是因为你现在可以在几秒内得到针对任何立场的有力反驳论证。更危险，是因为 AI 能生成听起来很有道理但实际上是胡说的内容。批判性思维帮助你把 AI 当作工具而非神谕。


## 自我评估量表

| 维度 | 新手（1） | 发展中（2） | 熟练（3） | 专家（4） |
|------|-----------|------------|----------|----------|
| **发现假设** | 没有帮助就无法识别假设 | 在提示下能找到明显的假设 | 能定期发现论点中的隐含假设 | 自动质疑他人忽视的根本假设 |
| **钢人论证** | 对对立观点进行稻草人攻击 | 能准确呈现对立观点 | 比大多数支持者更公平地呈现对立观点 | 能将对立论点强化到超越原始表述 |
| **因果推理** | 经常混淆相关性与因果性 | 在提示下能识别这一区别 | 能独立持续地应用这一区别 | 能识别微妙的混淆因素和替代解释 |
| **自我觉察** | 不知道自己的偏见 | 承认偏见存在但无法发现它们 | 能在反思中察觉自己的偏见 | 能在偏见形成时实时发现它们 |

## 结业条件

当你能做到以下事情时，说明你已掌握本模块：

- [ ] 不借助 AI，读完一篇新闻文章并列出 3 个以上隐含假设
- [ ] 写出一个你强烈反对的立场的钢人论证，并由他人确认其公平性
- [ ] 描述一件你改变了看法的事，并解释是什么推理让你改变了想法
- [ ] 在实时对话中察觉到自己正在做某个假设，并纠正它

**结业测试：** 找一篇为你本能反对的立场辩护的文章。写一篇 200 字的钢人论证。然后再写 200 字的批判。不使用任何 Prompt 完成。如果你的钢人论证能通过"公平性检验"（持有该观点的人会认为这是对他们立场的准确呈现），你就毕业了。
